# Issue Templates

## Bug report

Include:

- exact command
- full traceback/log output
- bundle path
- protocol version/hash
- `submission_review.json` if available

## Feature request

Include:

- problem statement
- expected user impact
- proposed API/contract changes
- migration and compatibility implications

## Reproducibility issue

Include:

- environment details
- rerun mismatch evidence
- paths to reproducibility artifacts

## Documentation issue

Include:

- current page URL/path
- exact confusing or missing section
- expected audience and desired outcome
