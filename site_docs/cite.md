# Cite

When citing AutoMechInterp, include enough metadata for reproducibility.

## Minimum citation payload

- repository URL
- commit hash or release tag
- protocol version/hash
- bundle identifiers

## Recommended supplemental metadata

- command used for evaluation
- rerun count and determinism status
- environment manifest path

## Example reporting sentence

“Claims were evaluated with AutoMechInterp (commit `<hash>`), protocol `<version/hash>`, and deterministic submission review (`reruns=3`), with environment manifest at `<path>`.”
